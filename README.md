# @cryptella/crypto

## Status

__WORK IN PROGRESS__

## What

This package implements [Web Crypto](https://developer.mozilla.org/en-US/docs/Web/API/Web_Crypto_API) helpers for __ECDSA__ signatures. Compatible with modern browsers, Bun and Node.js.

## Compatibility

- Bun 1+
- Node.js 16+
- Modern browsers
